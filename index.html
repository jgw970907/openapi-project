    웹사이트 소개

    경기도 사람들이 취업정보를 알아갈 수 있도록 제작한 웹사이트입니다.
    잡아바 어플라이가 제공하는 api 4가지를 활용하여 만든 웹사이트입니다. (html,css, javascript(+jquery)를 활용했습니다.)
    상단 헤더의 일자리 공고를 클릭시 경기도 내 고용중인 회사들의 공고를 짧게 확인 가능합니다.
    공고보기 클릭시 워크넷 홈페이지로 이동합니다. 해당 페이지는 페이지네이션으로 구성했습니다. 또한 검색이 가능합니다.
    두번째와 세번째는 모두 무한스크롤로 구성한 페이지입니다. 대외활동과 일자리 지원정책을 확인할 수 있습니다.
    네번째는 교육훈련과 관련된 웹페이지를 볼 수 있도록 구성했습니다. 해당 페이지는 더보기 버튼을 클릭해 내용을 볼 수 있도록 구성했습니다.
    (해당 웹사이트는 상업적인 목적을 위해 제작된 웹사이트가 아닙니다.)
